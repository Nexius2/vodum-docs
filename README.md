
# VODUM Documentation (GitHub Pages)

This repository contains the documentation website for **VODUM**, powered by **MkDocs Material**.

## Local preview

```bash
pip install mkdocs-material
mkdocs serve
```

Then open: http://127.0.0.1:8000

## Deploy on GitHub Pages

A workflow is included: `.github/workflows/docs.yml`

Make sure GitHub Pages is configured to use **GitHub Actions** as the source.
