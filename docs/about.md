**VODUM** is an advanced management tool for **Plex and Jellyfin**.

It provides a centralized interface to:
- manage users and subscriptions
- control access to servers and libraries
- automate tasks and maintenance
- monitor activity through logs
- configure mailing and notifications

VODUM is designed to simplify the daily management of media servers while keeping
everything **structured, automated, and traceable**.
